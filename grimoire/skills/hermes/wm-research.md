# WhiteMagic Research

Multi-source web research with enhanced fetching, rabbit hole exploration, and knowledge graph synthesis.

## When to Use

- Researching topics requiring multiple sources
- Competitive analysis and landscape mapping
- Technology evaluation
- Literature reviews

## How to Use

### Quick Web Search
```
wm(thought="search the web for MCP 2.0 specifications")
```

### Enhanced Fetch (with outline + chunking)
```
wm(route="gana_chariot.web_fetch_enhanced", args={"url": "..."})
```

### Rabbit Hole Research (multi-hop)
```
wm(thought="research the history of holographic memory systems")
```

### Store Research Findings
```
wm(thought="create memory: Research findings on <topic>")
```

## Workflow

1. Search existing memories for prior research
2. Web search for current information
3. Enhanced fetch key sources
4. Rabbit hole for depth (optional)
5. Extract knowledge graph from findings
6. Store synthesis in `research` galaxy
7. Cross-reference with existing knowledge
