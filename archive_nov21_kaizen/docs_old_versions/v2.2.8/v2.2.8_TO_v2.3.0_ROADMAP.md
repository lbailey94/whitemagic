## üí° Strategic Considerations

### Multi-Language Philosophy

**Python** (Core):

- Primary language
- Easiest for AI agents
- Rich ecosystem
- Rapid prototyping

**Rust** (Performance):

- Performance-critical paths
- Memory consolidation
- Search indexing
- Binary utilities

**Haskell** (Correctness):

- Type-safe transformations
- Symbolic reasoning
- State machines
- Query DSL

**TypeScript** (Web):

- MCP server
- Dashboard
- Website
- Browser extensions

**Go** (Infrastructure) [Optional]:

- CLI distribution
- gRPC services
- Microservices

### When to Use What

**Rule of thumb**:

1. **Start in Python** - Prototype, iterate quickly
2. **Optimize hot paths in Rust** - If 10x speedup needed
3. **Type-critical code in Haskell** - If correctness paramount
4. **Web/UI in TypeScript** - Modern, type-safe, ecosystem
5. **Infrastructure in Go** - If simple concurrency needed
