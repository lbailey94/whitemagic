## ğŸ“Š **Project Overview**

**WhiteMagic** is a production-ready tiered memory management system for AI agents, featuring native MCP (Model Context Protocol) integration for modern AI coding assistants.

### **Core Components**

1. **Python SDK** - Core memory management library
   - Tiered storage (short-term, long-term, archive)
   - Full-text search with tag filtering
   - Automatic consolidation and promotion
   - 40+ tests, 85% coverage

2. **MCP Server** - Native IDE integration
   - Published on npm: `whitemagic-mcp@2.1.0`
   - 7 tools + 4 resources
   - Works with Cursor, Windsurf, Claude Desktop
   - 27 automated tests (100% passing)

3. **FastAPI Backend** - REST API with authentication
   - JWT + API key auth
   - Rate limiting with Redis
   - PostgreSQL storage
   - Whop integration for monetization

4. **Dashboard** - Web interface for memory browsing
   - Vanilla JS + Tailwind CSS
   - Full CRUD operations
   - Real-time search and filtering
   - Deployment-ready for Vercel

5. **Docker Compose** - Complete local stack
   - API + Postgres + Redis + Dashboard + Caddy
   - One-command deployment
   - Automatic HTTPS with Caddy
