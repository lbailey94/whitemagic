# state.current

**Category**: session | **Safety**: read
**Gana**: `gana_heart`

## Description

Get the current work state snapshot — current task, active tasks, next steps, recent file modifications, and context. This is the primary tool to call on connecting to understand what to work on next.

```json

    "idempotency_key": {
      "type": "string",
      "description": "Optional idempotency key. For write tools, retries with the same key will replay prior results."
    },
    "dry_run": {
      "type": "boolean",

      "default": false
    },
    "now": {
      "type": "string",
      "description": "Optional ISO timestamp override for deterministic evaluation/replay (best-effort)."
    }
  }
}
```
