# windsurf.sync

**Category**: archaeology | **Safety**: write
**Gana**: `gana_chariot`

## Description

Full incremental pipeline: export all sessions → compare with previous exports → ingest only new/changed sessions → report. This is the recommended way to keep the sessions galaxy up to date.

    }
  }
}
