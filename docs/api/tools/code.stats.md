# code.stats

**Category**: system | **Safety**: read
**Gana**: `gana_chariot`

## Description

Get code structure graph statistics: node count, edge count, node types, edge types, last build time, project root, and Rust availability.

```json
    },
    "idempotency_key": {
      "type": "string",
      "description": "Optional idempotency key. For write tools, retries with the same key will replay prior results."
    },
    "dry_run": {
      "type": "boolean",

      "default": false
    },
    "now": {
      "type": "string",
      "description": "Optional ISO timestamp override for deterministic evaluation/replay (best-effort)."
    }
  }
}
```
