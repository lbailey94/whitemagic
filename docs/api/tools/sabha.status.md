# sabha.status

**Category**: governance | **Safety**: read
**Gana**: `gana_three_stars`

## Description

Get the collective vitality status of all 28 Ganas. Shows how many are healthy, degraded, silent, or struggling. Reports the overall saṃghāta (unity) score. Based on 12.108.31: 'Unity is the great refuge of the gaṇas.'
