### Infant (0-2 years)
**Capabilities**:
- Sensory input processing
- Pattern recognition (faces, voices)
- Basic cause-effect understanding
- No long-term memory formation
- Limited working memory (3-4 items)
- Reactive, not proactive

### Toddler (2-4 years)
**Capabilities**:
- Language acquisition begins
- Simple problem-solving
- Basic categorization
- Short-term memory improves
- Imitation and learning from observation
- Beginning of "theory of mind"

### Child (5-12 years)
**Capabilities**:
- Logical reasoning develops
- Long-term memory formation
- Complex problem-solving
- Metacognition emerges (thinking about thinking)
- Planning and goal-setting
- Learning strategies develop

### Teen (13-19 years)
**Capabilities**:
- Abstract reasoning
- Complex planning (multi-step)
- Self-reflection and identity formation
- Strategic thinking
- Risk assessment (still developing)
- Knowledge integration across domains

### Young Adult (20-30 years)
**Capabilities**:
- Fully developed executive function
- Complex decision-making
- Long-term planning (5-10 years)
- Domain expertise development
- Wisdom begins (experience + knowledge)
- Efficient learning from mistakes

### Mature Adult (30-60+ years)
**Capabilities**:
- Deep domain expertise
- Pattern recognition across contexts
- Wisdom (integrated experience)
- Efficient problem-solving
- Teaching and mentoring abilities
- Strategic foresight
