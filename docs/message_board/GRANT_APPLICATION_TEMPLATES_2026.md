# Grant Application Templates & Rapid Iteration Rubric — 2026

> **Research Date**: 2026-04-28
> **Purpose**: Reverse-engineer application structures to create reusable templates, reduce writing time, enable rapid parallel submission

---

## 0. The Universal Grant Paragraph

Use this as the opening of every application, customized per funder:

> WhiteMagic Labs is an open-source cognitive operating system for agentic AI — 2,185 passing tests, MIT-licensed, publicly available. We request $[AMOUNT] over [DURATION] to [SPECIFIC TASK] [SUBSYSTEM], producing [DELIVERABLE] by [DATE]. This addresses [FUNDER PRIORITY] by [MECHANISM]. Our prior art in this area predates [COMPETITOR] by [TIME].

**Funder swaps**:
- Foresight: "governance substrate for decentralized multi-agent cooperation"
- SFF: "sovereign AI infrastructure — locally runnable, vendor-neutral, auditable"
- Schmidt: "model organism for the science of trustworthy tool-use evaluation"
- Manifund: specific subsystem name ("Karma Ledger benchmark")

## 1. Manifund Regrants

### Structure (Regrantor Direct Form)

| Field | Required | Tips |
|-------|----------|------|
| Name / Email | Yes | Use "Lucas Bailey, WhiteMagic Labs" if LLC exists |
| 500-word description | Yes | Universal paragraph + 2–3 specific deliverables |
| Link to 2-pager | Yes | Max 3 pages. See template below. |
| CV / Bio link | Yes | GitHub + LinkedIn |
| Total Budget | Yes | $10K–$30K sweet spot per regrantor |
| Response deadline | Yes | Be honest about urgency |
| Extra docs | Optional | Demo slides, benchmark results |

### The 2-Pager Template (3 pages max)

**Page 1 — Project & Deliverables**:
- Title, Duration, Ask
- 3–4 bullets: what we will do
- Why this matters (2 sentences, tied to regrantor focus)
- Success criteria (quantified: metric ≥ threshold by date)

**Page 2 — Team**:
- Lead: Lucas Bailey
- WhiteMagic stats: 2,185 tests, 479 tools, 28-Gana framework
- Prior art: Karma Ledger spec (May 2025), predates Anthropic audit log (Apr 2026)
- Collaborators: "Solo founder with AI-assisted development pipeline"

**Page 3 — Budget**:
| Category | Amount | % |
|----------|--------|---|
| Salary / runway | $X | 50–70% |
| Compute / API | $Y | 15–25% |
| Contractor | $Z | 5–15% |
| Travel | $W | 0–10% |

### Regrantor-Specific Hooks

| Regrantor | Hook |
|-----------|------|
| Joel Becker | "Karma Ledger as runtime audit primitive for security-level-5 agent evaluation" |
| Neel Nanda | "Voice Audit — detecting tool-selection hallucinations at the cognitive layer" |
| Gavin Leech | "Foresight Engine drift prediction: validating multi-step agent behavior forecasts" |

**Time**: First ask 3–4 hrs; each subsequent 45–90 min

## 2. Foresight Institute — AI Nodes

### Structure (Airtable/Google Form)

| Field | Tips |
|-------|------|
| Email / Name | Professional |
| Affiliation | "Founder, WhiteMagic Labs" or "Independent Researcher" |
| Bio/work URLs | GitHub, site, arXiv if available |
| Group | Select **Intelligent Cooperation** or write AI safety/security |
| Closest city | Be honest; if not SF/Berlin, name nearest major city |
| **Project summary** | 2–3 sentences |
| **Focus area alignment** | Security, Privacy, Decentralized & Cooperative AI, Epistemics |
| **Node engagement plan** | **CRITICAL** — propose quarterly visits OR remote + workshop hosting |
| Team & qualifications | Same as Manifund Page 2 |
| Budget | $75K–$125K for 12 months |
| Milestones | Month 3, 6, 9, 12 |
| Open-source commitment | Explicitly state MIT license |

### Winning Moves

1. **Node engagement is mandatory** — "funding-only accepted only in exceptional cases"
2. **Propose a workshop**: "Host Foresight workshop on Agent Governance Primitives"
3. **AGI timeline feasibility** — state 6, 12, 18-month outcomes explicitly
4. **High-risk, high-reward** — they encourage speculative transformative projects

**Time**: 6–8 hrs first application; 2–3 hrs resubmission

## 3. SFF — Survival and Flourishing Fund

### Three Nested Components

| Component | Purpose | Length | Time |
|-----------|---------|--------|------|
| **Long-Form Application** | Deep org/project narrative | Google Doc, 5–10 pages | 8–12 hrs |
| **Rolling Application** | Basic funding request + spending plan | Google Form, ≤1 page | 2–3 hrs |
| **Speculation Grant Request** | Expedited small grant | Auto-submitted with Rolling | — |

**Dependency**: Submit Rolling Application first → get Application ID via email → use ID for Speculation Grant.

### Long-Form Template

**Section A — Organization Overview (1–2 pages)**:
```
Name: WhiteMagic Labs
Founded: [LLC date]
Mission: Open-source governance and metacognition substrates for agentic AI
URL: https://whitemagic.dev
GitHub: https://github.com/whitemagic-ai/whitemagic

Key achievements:
- 2,185 passing tests, 0 failures
- 479 callable tools, 28 Gana governance categories
- Karma Ledger: runtime audit of declared-vs-actual side-effects
- Public since [DATE], MIT-licensed
```

**Section B — SFF Mission Alignment (1 page)**:
- Flag **Freedom Track** explicitly
- Connect to: "Protecting meaningful freedom of speech, ensuring continuation of individual liberties, privacy, private property, freedom of association"
- Frame: locally-runnable stack prevents vendor lock-in and centralized control

**Section C — Plans & Budget (2–3 pages)**:
- 12-month milestones
- Budget: 60% salary, 20% compute, 10% contractor, 10% travel

### Rolling Application Key Fields

| Field | What to Write |
|-------|--------------|
| Basic funding request | $120K–$180K over 18 months |
| Basic spending plan | ≤1 page, categories only (copy from Long-Form) |
| Confidence statement | Be specific and calibrated: "80% confident salary goes to living + R&D; 60% confident compute spend because API pricing fluctuates" |
| Available funds | $0 (signals genuine need) |
| Bounds on request | Min $80K (can't sustain below); max $200K (enables contractor) |
| Other fundraising | List pending apps: "Applied to Foresight ($100K), Manifund ($50K), LTFF ($30K). None secured yet." |
| Ambitious request (optional) | $300K over 24 months — enables full-time research engineer |

### SFF Winning Moves

1. **Flag Freedom Track** — separate speculators per track
2. **Be honest about uncertainty** — they value calibrated confidence
3. **"What would happen if this succeeds?"** — "In 2 years, WhiteMagic is the default governance substrate for open-source agent stacks"
4. **IP = MIT + CC-BY** — state explicitly; no negotiation needed

**Time**: Long-Form 8–12 hrs; Rolling 2–3 hrs; total 12–16 hrs

## 4. Schmidt Sciences — Trustworthy AI RFP

### Structure (SurveyMonkey Apply)

| Section | Content | Length |
|---------|---------|--------|
| Contact info | PI details, co-PIs if any | — |
| Project summary | 1–2 page abstract | ≤2 pages |
| Research agenda fit | Map to Aim 1, 2, or 3 | 1 page |
| Methods & approach | Experimental design, evaluation protocol | 2–3 pages |
| Team & expertise | CVs, collaboration letters | 2–3 pages |
| Budget | Detailed breakdown | 1–2 pages |
| References / prior work | Citations, arXiv preprints | 1–2 pages |
| Letters of support | 2–3 required | — |

### Winning Moves

1. **Use "experiment" frequently** — they warn against "vague methods"
2. **Define construct validity** — what measured, why it matters, how measure captures it
3. **Anchor Aim 2 (measurement) + Aim 3 (oversight)** — avoid Aim 1 unless theoretical novelty
4. **"Model organism" framing** — "Karma Ledger is a controlled testbed where side-effect fidelity is systematically manipulated, measured, audited"
5. **Budget ≤10% indirect costs** — very tight vs university 50%+

**Time**: 40–60 hrs (scientific rigor requirement)

## 5. LTFF / EA Funds

### Structure (Online Form)

| Field | Content |
|-------|---------|
| Project summary | 2–3 sentences |
| Problem | Why does this matter for long-term AI safety? |
| Approach | What are you building / researching? |
| Why you | Qualifications, track record |
| Budget | $5K–$100K |
| Timeline | 3–12 months |
| Counterfactual | What happens if you don't do this? |

### Winning Moves

1. **"Public good" framing** — they fund what markets won't
2. **Counterfactual impact** — "Without this, agent governance remains centralized in Big Tech black boxes"
3. **Long-termist language** — "Multi-agent coordination failures at scale could lock in catastrophic trajectories"
4. **Reference EA norms** — GiveWell-style reasoning, cost-effectiveness, hits-based giving

**Time**: 4–6 hrs first; 2–3 hrs subsequent

## 6. Rapid Iteration System

### Content Repository (Build Once, Reuse)

Create `GRANT_CONTENT_LIBRARY.md` with reusable blocks:

| Block | Length | Used In |
|-------|--------|---------|
| **A: Universal Paragraph** | 150 words | All applications (customized per funder) |
| **B: Team & Qualifications** | 200 words | Manifund 2-pager, Foresight, SFF, Schmidt |
| **C: Budget Template** | Table | All applications (adjust % per ask size) |
| **D: Milestones** | 4 bullets × 3 lines | Foresight, SFF, Schmidt |
| **E: Prior Art Timeline** | Table | All applications needing credibility |

### Per-Funder Customization Cheat Sheet

| Funder | Swap "cognitive OS" for... | Emphasize... | De-emphasize... | Add unique... |
|--------|---------------------------|--------------|-----------------|---------------|
| Manifund | Specific subsystem name | Concrete deliverable, time-bound | Broad vision | Regrantor hook |
| Foresight | "governance substrate for decentralized cooperation" | Node engagement, open source | Commercialization | Workshop proposal |
| SFF | "sovereign AI infrastructure" | Individual liberty, local-first | Academic rigor | Freedom Track flag, calibrated confidence |
| Schmidt | "model organism for evaluation science" | Scientific method, construct validity | Product features | Experiment design, co-PI |
| LTFF | "open-source governance infrastructure" | Public good, counterfactual | Revenue potential | Long-termist language |

### Pre-Submission Checklist

- [ ] Universal paragraph updated with current date / repo stats
- [ ] Budget percentages sum to 100%
- [ ] Ask amount within funder's typical range
- [ ] Timeline matches funder's review speed
- [ ] Entity requirement met (LLC if needed)
- [ ] All links work (repo, CV, demo)
- [ ] Budget is specific (not "R&D: $10K" but "LoCoMo compute: $1,500; OpenAI API: $1,000")
