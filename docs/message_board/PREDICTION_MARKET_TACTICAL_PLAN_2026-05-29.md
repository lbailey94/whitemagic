## Part 3: Bankroll & Risk Management

### The Kelly Framework

The Kelly Criterion gives optimal bet size:

```
f* = (bp - q) / b

where b = net odds (payout / stake), p = true probability, q = 1-p
```

**Example:** Market prices a contract at $0.40 (40% implied). Your true probability is 70%.
- b = (1 - 0.40) / 0.40 = 1.5
- f* = (1.5 * 0.70 - 0.30) / 1.5 = 0.50 → **50% of bankroll (full Kelly)**
- **Quarter-Kelly = 12.5%** — this is what professionals actually use

## Appendix: Quick Reference

### Kelly Calculator
```python
def kelly(market_price, true_probability):
    b = (1 - market_price) / market_price
    p = true_probability
    q = 1 - p
    return (b * p - q) / b

# Example: market 0.60, true prob 0.85
full = kelly(0.60, 0.85)      # 0.417 → 41.7%
quarter = full / 4            # 0.104 → 10.4% of bankroll
```

### Edge Threshold Rule
```
Trade only if: |true_probability - market_price| ≥ 0.15
And your confidence in the true_probability ≥ 0.80
```

### Correlation Check
```
Before any trade, ask: "What single event would hurt this position the most?"
If that same event would hurt >15% of your portfolio → reduce size by 50%.
```
