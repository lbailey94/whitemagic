# WhiteMagic - Quick Start Guide

## What Is It?
Memory system for AI agents. Like RAM for your AI assistant - remembers stuff between conversations.

## How It Works
1. AI talks to you
2. WhiteMagic saves important parts
3. AI recalls it next time

## Why Use It?
- AI remembers your preferences
- No repeating yourself
- Better conversations
- Search past chats

## Versions

**Free (CLI)**: Local only, unlimited, $0
**Pro ($9/mo)**: Cloud API, 10K requests/day, 10GB
**Enterprise ($49/mo)**: 100K requests/day, 100GB, team features

## Revenue Model
- Free tier builds community
- Paid tiers for cloud hosting
- No data selling
- Usage-based pricing

## Get Started
```bash
pip install whitemagic
whitemagic init
```
